Using selenium generate an automated bot that logs into instagram finds the followers of the targeted
account and follows all the followers

InstaFollower:

    contains InstaFollower class InstaFollower constructor takes three arguments user_name,
    password of an instagram account and webdriver 

    methods:
        login: login to instagram
    
        find_followers: takes targeted account user_name as string

        follow : follows all the followers of the target account
